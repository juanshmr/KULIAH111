## Deskripsi Singkat
Aplikasi ini digunakan untuk menghitung gaji karyawan berdasarkan:
- Jam kerja per minggu  
- Upah per jam  
- Status karyawan  

Ketentuan pajak:
- **Karyawan Tetap** → Pajak 10%  
- **Karyawan Kontrak** → Pajak 5%  

Output yang ditampilkan:
- Nama karyawan  
- Gaji kotor  
- Pajak  
- Gaji bersih
